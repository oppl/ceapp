== Ergebnis ==
Das Ergebnis dieses Projektes ist die funktionsfähige Applikation „Clash eliminator“ welche alle musst-haves der
SW-Spezifikation enthält. Zusätzlich wurden auch einige nice-to-haves umgesetzt. Wie bereits zu Beginn vorgesehen, kann
die Applikation für weitere Projekte entsprechend adaptiert bzw. erweitert werden.
Zusätzlich wurde ein Betriebshandbuch erstellt, welches die wesentlichen Funktionen der Applikation veranschaulicht.
